# ⛏️ Mining Dashboard — Power BI

> A collection of interactive **Power BI dashboards** built for mining operations monitoring, covering mine planning, reclamation tracking, and production reporting across mining sites in Indonesia.

---

## 📖 About This Project

This repository contains Power BI (`.pbix`) dashboard files developed to support data-driven decision making in the mining industry. Each dashboard focuses on a specific operational aspect — from mine planning and production tracking to environmental reclamation monitoring.

The dashboards are designed to provide clear, visual insights into key mining metrics, enabling stakeholders to monitor performance and progress efficiently.

---

## 📁 Repository Structure

```
mining-dashboard/
│
├── Mine Plan Rembang.pbix        ← Mine planning dashboard for Rembang site
├── Mining Dashboard.pbix         ← General mining operations dashboard
├── PBI REKLAMASI FIX.pbix        ← Land reclamation monitoring dashboard
└── PBI STEAMROCK.pbix            ← Steamrock production dashboard
```

---

## 📊 Dashboard Overview

| File | Description |
|------|-------------|
| `Mine Plan Rembang.pbix` | Visualizes mine planning data for the **Rembang** mining site, including extraction schedules, volume targets, and plan vs. actual comparisons |
| `Mining Dashboard.pbix` | General-purpose mining operations dashboard covering key performance indicators (KPIs) such as production volume, efficiency, and operational metrics |
| `PBI REKLAMASI FIX.pbix` | Monitors **land reclamation** progress — tracking restored areas, compliance status, and reclamation milestones post-mining |
| `PBI STEAMROCK.pbix` | Tracks **Steamrock** activity and production data, including material handling and performance metrics |

## 🚀 Getting Started

### Prerequisites

Make sure you have **Microsoft Power BI Desktop** installed.
Download it for free at: [https://powerbi.microsoft.com/desktop](https://powerbi.microsoft.com/desktop)

### How to Open

1. **Clone this repository**
   ```bash
   git clone https://github.com/Chelseaayu/mining-dashboard.git
   cd mining-dashboard
   ```

2. **Open any `.pbix` file** with Power BI Desktop by double-clicking or via:
   ```
   File → Open → Browse to the .pbix file
   ```

3. **Explore the dashboard** — use slicers and filters to interact with the data.

> ⚠️ **Note:** Some dashboards may require reconnecting to the original data source. Go to `Home → Transform Data → Data Source Settings` to update the connection if needed.
